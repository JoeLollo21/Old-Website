Creative/Web:

Mapping Seattle's Catholic Churches:

  • A fun little interactive map using the little bit of geospatial data visualization skills I have. Click each data point to learn about which church is there and its addresss.
  • I made this using the Geocoder package for R, and web-scraped the data from the Archdiocese of Seattle's Parish Finder.

Generators: Created with a mix of web and scripting languages.

  • East/West Bowl Player Name Generator: Based on the Key & Peele sketch (hosted on Perchance).

Bots and Other Code:

Lots of these come from my Web Programming class from UW Bothell, which this website was originally created for.

  • Ryanator: A simple word replacement Google Chrome Extension in JavaScript, based on Jonah Ryan from the TV series Veep.
  • ChadBot: A Discord bot version of Chad from Saturday Night Live, that responds "ok" any time someone says "ok," "okay," or "k".
  • RStudio Synthwave Theme: A custom RStudio theme I made.
